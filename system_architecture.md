# System Architecture

The system consists of four major components.

## 1. Frontend
The frontend provides the user interface where users can:
- Register
- Login
- Upload invoices
- View extracted data

## 2. Backend
The backend handles:
- User authentication
- File processing
- Communication with the database

## 3. Database
The database stores:
- User information
- Uploaded invoice records
- Extracted invoice data

## 4. OCR Module
The OCR module extracts text from uploaded invoices using OCR technology.
