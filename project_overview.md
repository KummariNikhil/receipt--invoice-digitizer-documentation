# Project Overview

The Receipt and Invoice Digitizer is a web-based application that converts physical receipts and invoices into structured digital data.

Users can upload receipts in PDF, image, or scanned format. The system extracts important information from the documents and stores it in a database.

## Features
- User registration and login
- Upload receipts and invoices
- Extract important invoice details
- Store extracted data securely
- Retrieve and analyze stored records
