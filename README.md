# Receipt & Invoice Digitizer – Documentation

This repository contains the documentation for the Receipt and Invoice Digitizer project developed during the Infosys Springboard Virtual Internship.

## Project Description
The Receipt & Invoice Digitizer is a web application that allows users to upload receipts and invoices in PDF or image format and convert them into structured digital data.
