# API Documentation

The backend provides APIs for communication between frontend and server.

## 1. Register User
Endpoint: /register  
Method: POST  
Description: Registers a new user.

## 2. Login User
Endpoint: /login  
Method: POST  
Description: Authenticates user and returns a token.

## 3. Upload Invoice
Endpoint: /upload  
Method: POST  
Description: Uploads invoice file and processes it.

## 4. Get Invoice Data
Endpoint: /invoices  
Method: GET  
Description: Retrieves stored invoice records.
