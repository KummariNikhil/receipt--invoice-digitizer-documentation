# User Manual

## Step 1: Register
The user creates an account by entering name, email, and password.

## Step 2: Login
The user logs into the system using email and password.

## Step 3: Upload Invoice
The user uploads an invoice in PDF or image format.

## Step 4: Processing
The system processes the invoice and extracts important information.

## Step 5: View Results
The extracted invoice data is displayed and stored in the database.
